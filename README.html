ProR - Rodin Integration

This project provides an integration for ProR and Rodin, both Eclipse-based applications:

  • ProR is a platform for Requirements Engineering, based on the RIF/ReqIF standard.
  • Rodin is a platform for formal modeling using the Event-B formal method.

To use this project, you also need the ProR and Rodin source code. The details are documented on the ProR Website.
